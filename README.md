# TechInspired_EnuguSummit
## Overview
______________________________________
 This is a sentiment analysis of user tweets for the biggest tech event in the year 2022 in Enugu. I made use of python,excel and powerBi for the overall data extraction,cleaning,analysis and data visualization. Use this link to view full documentation.
